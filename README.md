## Development

```bash
pnpm dlx degit importantimport/a2z && cd a2z
pnpm dev
```

### Environment

| Name           | Default                         |
| -------------- | ------------------------------- |
| A2Z_RPC_URL    | `http://localhost:6800/jsonrpc` |
| A2Z_RPC_SECRET | `undefined`                     |
